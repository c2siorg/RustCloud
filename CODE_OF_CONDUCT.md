# Code of Conduct

## Our Pledge

We want RustCloud to be a welcoming project for everyone, regardless of experience level, background, or identity. Whether you're opening your first issue or reviewing a complex PR, you should feel comfortable participating.

## Expected Behavior

- Be respectful and constructive in code reviews and discussions
- Welcome newcomers and help them get oriented
- Accept feedback gracefully — disagreements on technical direction happen, but keep them focused on the code, not the person
- Give credit where it's due

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Dismissing or belittling contributions without explanation
- Publishing others' private information without consent
- Any behavior that would make someone feel unwelcome or unsafe

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, the C2SI Slack (`#rust-cloud`), and any other venue where RustCloud is being discussed in an official capacity.

## Enforcement

If you witness or experience behavior that violates this Code of Conduct, please reach out to a project maintainer directly via Slack or email. Reports will be handled confidentially. Maintainers are responsible for clarifying acceptable behavior and taking appropriate action, which may include a warning or removal from the project.
